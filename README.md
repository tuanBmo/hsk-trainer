# HSK 3.0 Trainer - Mastery Edition

Ứng dụng hỗ trợ học từ vựng HSK từ cấp độ 1 đến 9 với giao diện hiện đại, vui nhộn.

## ✨ Tính năng
- Học qua 3 chế độ: Nghĩa, Pinyin, Chữ Hán.
- Hệ thống "HP" (máu) giúp tăng độ kịch tính khi học.
- Sổ tay lưu lại các từ vựng đã chọn sai để ôn tập.
- Hỗ trợ dữ liệu từ file Excel (.xlsx).

## 🚀 Hướng dẫn cài đặt
1. Cài đặt Python (phiên bản 3.10 trở lên).
2. Clone project: `git clone https://github.com/user/hsk-trainer.git`
3. Cài đặt thư viện: `pip install -r requirements.txt`
4. Chạy ứng dụng: `python main.py`

## 📊 Định dạng dữ liệu
Các file level (ví dụ: `HSK1.xlsx`) cần có các cột theo thứ tự:
- Cột A: Chữ Hán (Hanzi)
- Cột B: Phiên âm (Pinyin)
- Cột C: Nghĩa (Meaning)
- Cột D: Loại từ (Word Type)
- Cột E: Ví dụ (Example)
